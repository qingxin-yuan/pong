# Pong Game

A basic pong game using SVGs.

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a: up
* z: down
* f: fire the ball after goal

**Player 2:**
* ▲ : up
* ▼: down
* enter: fire the ball after goal
